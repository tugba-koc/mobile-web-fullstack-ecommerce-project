# Expo Fullstack E-Commerce Project

A fullstack e-commerce app with a React admin panel, React Native mobile app, and Node.js backend.

## Structure

```
├── admin/      # React + TypeScript admin dashboard (Vite)
├── mobile/     # React Native mobile app (Expo)
└── backend/    # Express.js REST API
```

## Getting Started

### Backend
```bash
cd backend
npm install
npm start
```

### Admin
```bash
cd admin
npm install
npm run dev
```

### Mobile
```bash
cd mobile
npm install
npx expo start
```

## Tech Stack

| Layer | Stack |
|-------|-------|
| Admin | React 19, TypeScript, Vite |
| Mobile | React Native, Expo 54, Expo Router |
| Backend | Node.js, Express 5, dotenv |
